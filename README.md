StudyHub est une application éducative développée en Python avec Tkinter, conçue pour faciliter l’apprentissage de la Programmation Orientée Objet (POO).
Elle propose :

-Des quizz interactifs pour tester et renforcer les connaissances.

-Un suivi de progression avec score, niveaux et séries de jours.

-Des formations modulaires : Fondamentaux Python, POO, Python Avancé.

-Une gamification motivante avec points, niveaux et déblocage de contenu.

-Un centre d’aide et des défis de code pour pratiquer concrètement.

L’application vise à rendre l’apprentissage plus ludique, progressif et personnalisé pour chaque utilisateur.
